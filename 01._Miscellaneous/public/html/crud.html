This page documents what I have learned about applying CRUD in node js.

Hub

1. Using an array to store data

Instead of using a real database, using a simple array is fine for prototyping/testing


            let movies = [
                { id: 1, title: 'Interstellar', year: 2014 },
                { id: 2, title: 'Marty Supreme', year: 2025 }
            ];

            let nexId = 3;
Here is an array of movies with 3 properties: id, title and year.
nextId is used to ensure that every new movie added gets a unique id. Since id 1 and 2 is already taken, it starts at 3.

2. Applying CRUD pt.1


            GET /movies - gets movie(s)

            GET /movies{id} - gets specific movie

            POST /movies{id} - adds a new movie 

            PUT /movies{id} - modifies all fields

            PATCH /movies{id} - modifies the selected field(s)

            DELETE /movies{id} - removes a movie
Using {id} tells the server which movie is targeted.

3. Applying CRUD pt.2


            app.get("/movies", (req, res) => {
                res.send({ data: movies });
            });
            
            app.get("/movies/:id", (req, res) => {
                const providedMovieId = Number(req.params.id);
                const foundMovie = movies.find((movie) => movie.id === providedMovieId);
            
                if (!foundMovie) {
                return res.status(404).send({ errorMessage: `No movie found by id: ${req.params.id}` });
                }
                res.send(foundMovie);
            });
GET /movies returns the movie array
GET /movies{id} fetches specifc movie 


            app.post('/movies', (req, res) => {
                if (!req.body.title) {
                return res.status(400).send({ errorMessage: `Title is required`});
            };
            
            const newMovie = {
                id: nextId++,
                title: req.body.title,
                year: req.body.year
            };
            
            movies.push(newMovie)
            res.status(201).send({data: newMovie})
            });
Checks for title: if not, send back 400 (bad request)
If title is found, create newMovie object with id(nextId++ to auto-increment), title and year
Pushes the movie to the movie array and sends back what was created 


            app.patch('/movies/:id', (req, res) => {
                const providedMovieId = Number(req.params.id);
                const foundMovie = movies.find((movie) => movie.id === providedMovieId);
            
                if(!foundMovie) {
                    return res.status(404).send({ errorMessage: `No movie found by id: ${req.params.id}`})
                }
            
                if (req.body.title) foundMovie.title = req.body.title;
                if (req.body.year) foundMovie.year = req.body.year;
            
                res.status(200).send({ data: foundMovie });
            });
Searches for movie with specific id, if not found: returns 404 with error message
If found: checks if title and year was provided in body, then overwrites
Sends back updated movie with 200 


            app.delete('/movies/:id', (req, res) => {
                const providedMovieId = Number(req.params.id);
                const foundMovie = movies.find((movie) => movie.id === providedMovieId);
            
                if (!foundMovie) {
                return res.status(404).send({ errorMessage: `No movie found by id: ${req.params.id}` });
                }
            
                movies = movies.filter((movie) => movie.id !== providedMovieId);
                res.status(204).send();
            });
Same idea as above.
Using filter method to make a new array that doesn't contain the movie that was deleted

4. How to use it?

The typical way to see if it works is by using Postman.
In Postman you could write: 


            GET:
            http://localhost:8080/movies (get all movies)
