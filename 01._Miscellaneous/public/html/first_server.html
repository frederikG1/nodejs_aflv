1. Setting up the environment

Before making a lot of different files, there are a couple of commands to run in the terminal to set up your node environment properly.

The very first think to do is actually making a folder and making sure you are in the right place


            mkdir myapp
            cd myapp
This ensures that you are in the right place for when you want to use the following commands


            npm init
This creates a package.json file for you application


            npm i express
This installs express. Express is a framework for node.js that makes building web applications more simple

2. How to set it up?

After installing express, you need to "add" it through JavaScript


            const express = require('express');
This imports express


            const app = express();
And this instantiates express


            import express from 'express';
This is another solution, but with this you need "type": "module" in package.json

3. How to actually use it

The following is a simple way to get a server running locally


            const express = require ('express');
            
            const app = express();
            
            app.get('/', (req, res) => {
            res.send('Hello World');
            });
            
            app.listen(8080, () => {
            console.log('Server is running on http://localhost:8080');
            });
This will make a server on localhost:8080 displaying "Hello World"

4. Making use of routing

Routing is essentially what you input after "('/')". An example could be


            app.get('/energydrinks', (req, res) => {
                res.send('I like Redbull');
            });
When you enter localhost:8080/energydrinks it will display "I like Redbull".

You can also use a method called sendFile like this


            app.get('/energydrinks', (req, res) => {
                res.sendFile(__dirname + '/energydrinks.html');
            });
__dirname gives the correct path to the folder where your app.js(typical name) is located and then node sends the file to the browser 


            app.get('/energydrinks/:brand/:size', (req, res) => {
                res.send({ data: `
                    Your ${req.params.brand} is very nice.
                    What size it it ${req.params.size}?
                    `});
            });
